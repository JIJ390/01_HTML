li(List Item) 태그 : 목록의 한 줄

ol(Ordered List) 태그 : 순서가 있는 목록

			- type 속성 : 표시되는 숫자 형식 지정

			type="a" : 영어 소문자
			type="A" : 영어 대문자
			type="i" : 로마 소문자(i,ii,iii, ...v,...x)
			type="I" : 로마 대문자(I,II,III, ...V,...X)

			start 속성 : 목록 시작 번호
			reversed : 순서 뒤집기(역순, 시작부터 -1씩)

Emmet : HTML, CSS 빠른 자동완성을 지원하는 플러그인

			태그명 : 태그 자동 완성

			> : 태그 내부에 작성(하위)

			+ : 태그 옆에(형제)

			* 숫자 : 숫자만큼 생성(곱하기)에스터리스크
			
			{} : 태그 내부에 작성될 내용

			$ : 1 부터 1씩 증가하는 숫자
